link:: https://wikipedia.com :: Enlace de Prueba #Introducción Este proyecto es un simple compilador de texto plano en un archivo html para estilizarlo con lo basico. linea: #Como usar: Se puede adaptar este formato a cualquier archivo html solo agregando la siguiente linea en el head: "#script src='https://0-jsgh.github.io/format-text/meta/script.js'>#/script>" Reemplazar los # por "<". De manera dinamica tambien se conectará la hoja de estilos vinculada a el comportamiento del javascript, por lo que quedaria listo para implementar. linea: ##Opciones: se pueden agregar tres tipos de titulos: #Titulo 1 ##Titulo 2 ###Titulo 3 Cada uno de estos se asignan con los prefijos "#", "##" y "###" respectivamente. linea: ##Enlaces los enlaces se asignan con el prefijo "link" al inicio de una linea, seguido del enlace y el texto visible (link :: enlace.com :: texto). link:: meta/estilo.js :: script de archivo link::meta/style.css :: hoja de estilos linea: ##Videos se pueden insertar videos de Youtube con el prefijo video: al inicio de una linea, hay que poner el enlace de la siguiente manera: "https://youyube.com/embed/identificador-del-video" video:https://www.youtube.com/embed/rMjmAraA9H0?si=A2oWbCZlT4ndg2PN linea: ##Imagenes se pueden insertar 3 tipos de tamaños: normal "img:ruta/imagen.jpg", que tendra un tamaño fijo de 720 pixeles. img:https://upload.wikimedia.org/wikipedia/commons/7/79/%C3%81tomo.jpg pequeño "imgmin:ruta/imagen.jpg", que tendra un tamaño maximo de 360 pixeles. imgmin:https://upload.wikimedia.org/wikipedia/commons/7/79/%C3%81tomo.jpg grande "imgmax:ruta/imagen.jpg", que tendra un tamaño fijo de de 1280 pixeles. imgmax:https://upload.wikimedia.org/wikipedia/commons/7/79/%C3%81tomo.jpg si la pantalla se hace pequeña las imagenes se adaptaran al tamaño de pantalla.
